# Реализация UI: Приветствие и главное меню

## ✅ Реализованная функциональность

### 1. Окно приветствия
- **ASCII art заставка** — красочное приветствие с названием игры
- **Автоматический переход** к главному меню после нажатия Enter
- **Цветовая схема** — поддержка colorama для цветного вывода

### 2. Главное меню
- **5 пунктов меню**:
  1. Новая игра
  2. Загрузка  
  3. Настройки
  4. Моды
  5. Выход
- **Нумерованная навигация** — пользователь вводит только цифру
- **Валидация ввода** — проверка диапазона значений
- **Циклическое меню** — возвращение после каждого действия

### 3. Техническая реализация

#### Архитектура
```
src/
├── __init__.py
├── main.py              # Точка входа
└── dnd_mud/
    ├── __init__.py
    ├── core/
    │   ├── __init__.py
    │   ├── game.py          # Game класс
    │   ├── config.py        # Config
    │   └── exceptions.py    # Исключения
    └── ui/
        ├── __init__.py
        ├── console.py       # Базовый UI с цветами
        ├── screens.py       # Экраны приветствия и меню
        ├── menus.py         # Заготовка под меню
        └── components.py    # Заготовка под компоненты
```

#### Ключевые классы
- **Console** — консольный вывод с цветовой схемой
- **WelcomeScreen** — экран приветствия с ASCII art
- **MainMenuScreen** — главное меню с нумерованной навигацией
- **Game** — основной игровой цикл и обработка выбора

## 🚀 Запуск приложения

### Требования
- Python 3.12+
- Виртуальное окружение

### Установка зависимостей
```bash
# Активация виртуального окружения
source .venv/bin/activate

# Установка зависимостей
pip install PyYAML colorama
```

### Запуск
```bash
# Из корневой директории проекта
source .venv/bin/activate
python src/main.py
```

## 🎮 Демонстрация работы

При запуске приложения:
1. Показывается ASCII art заставка
2. После нажатия Enter появляется главное меню
3. Пользователь выбирает пункт меню вводом цифры 1-5
4. Для неверного ввода показывается ошибка
5. Для выхода из программы выбирается пункт 5
