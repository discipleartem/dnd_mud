# Руководство пользователя

## 🎮 Добро пожаловать в D&D Text MUD!

Это руководство поможет вам освоить консольную текстовую игру по мотивам Dungeons & Dragons 5e.

---

## 🚀 Быстрый старт

### Установка
```bash
# Клонирование репозитория
git clone https://github.com/discipleartem/dnd_mud.git
cd dnd_mud

# Создание виртуального окружения
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или .venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -e .

# Запуск игры
dnd-mud
```

### Первые шаги
1. Запустите игру командой `dnd-mud`
2. Посмотрите приветственное окно
3. Выберите "Создание персонажа"
4. Следуйте инструкциям на экране

---

## 📋 Главное меню

### Пункты меню
- **Новая игра** - начать приключение
- **Создание персонажа** - создать нового героя
- **Загрузка игры** - продолжить сохранение
- **Настройки** - язык, hardcore режим
- **Моды** - управление модификациями
- **Выход** - завершить игру

### Навигация
- Используйте цифры для выбора пунктов
- Enter для подтверждения
- Esc для возврата в предыдущее меню

## 🎭 Создание персонажа

### Шаг 1: Имя персонажа
- До 20 символов
- Разрешены буквы, дефис, пробелы
- Не может быть пустым

**Примеры:**
- ✅ `Арториус`
- ✅ `Джон Сноу`
- ❌ `123`
- ❌ `@#$%`

### Шаг 2: Раса и подраса
**Доступные расы:**
- **Человек** (базовый и вариант)
- **Эльф** (высокий, лесной, дроу)
- **Дворф** (холмовой, горный)
- **Полуорк**

### Шаг 3: Характеристики
**Методы генерации:**
1. **Стандартный** - выбор из [15, 14, 13, 12, 10, 8]
2. **Point Buy** - 27 очков для распределения
3. **Случайный** - 4d6 drop lowest

**Характеристики:**
- Сила (Strength)
- Ловкость (Dexterity)
- Телосложение (Constitution)
- Интеллект (Intelligence)
- Мудрость (Wisdom)
- Харизма (Charisma)

### Шаг 4: Класс
**Воинные классы:**
- **Воин** - мастер оружия и доспехов
- **Варвар** - ярость и первобытная сила

**Магические классы:**
- **Бард** - музыка и магия
- **Жрец** - божественная сила

**Специалисты:**
- **Плут** - скрытность и хитрость

### Шаг 5: Предыстория
- **Аколит** - служитель храма
- **Воин** - ветеран битв
- **Крестьянин** - простолюдин
- **Ученый** - исследователь знаний
- И другие...

### Шаг 6: Навыки
**Примеры навыков:**
- Акробатика, Атлетика
- Пронзательность, Запугивание
- Магия, История
- Выживание, Скрытность

### Шаг 7: Языки
- **Общий** (Common) - базовый язык
- **Эльфийский** - язык эльфов
- **Дворфийский** - язык дварфов
- **Орочий** - язык орков

### Шаг 8: Заклинания (для магов)
- **Заговоры** - малые заклинания
- **Заклинания** - основные магии
- **Слоты** - количество использований

## ⚔️ Игровые механики

### Боевая система
- **Инициатива** - порядок хода
- **Атака** - бросок d20 + модификатор
- **Урон** - зависит от оружия
- **Защита** - Класс Доспеха (AC)

### Характеристики
**Модификаторы:**
- 1 = -5, 2-3 = -4, 4-5 = -3
- 6-7 = -2, 8-9 = -1, 10-11 = +0
- 12-13 = +1, 14-15 = +2, 16-17 = +3
- 18-19 = +4, 20 = +5

### Уровни и опыт
- **Уровни 1-10** - лимит игры
- **Новые способности** - при повышении уровня
- **Увеличение характеристик** - на определенных уровнях

## 🎯 Команды в игре

### Основные команды
```
/status      - показать статус персонажа
/inventory   - показать инвентарь
/help        - список команд
/look        - осмотреться
```

### Команды в бою
```
/attack      - атаковать врага
/defend      - защититься
/cast <spell> - применить заклинание
/run         - попытаться убежать
```

### Социальные команды
```
/say <text>  - сказать что-то
/yell <text> - крикнуть
/whisper <name> <text> - шепнуть
```

## 🐛 Устранение проблем

### Частые вопросы

**Q: Игра не запускается**
A: Проверьте установку Python 3.12+ и активацию виртуального окружения

**Q: Пропал звук**
A: В консольной версии звука нет, только текст

**Q: Как изменить язык?**
A: Главное меню → Настройки → Язык

**Q: Где находятся сохранения?**
A: В папке `saves/` в директории игры

**Q: Можно ли изменить персонажа?**
A: Только созданием нового персонажа
