# Реализация UI: Приветствие и главное меню

## ✅ Реализованная функциональность

### 1. Окно приветствия
- **Простой приветственный экран** — минималистичное приветствие
- **Автоматический переход** к главному меню после нажатия Enter
- **Цветовая схема** — поддержка colorama для цветного вывода

### 2. Главное меню
- **5 пунктов меню**:
  1. Новая игра
  2. Загрузка  
  3. Настройки
  4. Моды
  5. Выход
- **Нумерованная навигация** — пользователь вводит только цифру
- **Валидация ввода** — проверка диапазона значений
- **Циклическое меню** — возвращение после каждого действия

### 3. Техническая реализация

#### Архитектура (KISS - Keep It Simple, Stupid)
```
src/
├── __init__.py
├── main.py              # Точка входа
├── core/               # Бизнес-логика
│   ├── game.py         # Game класс (координатор)
│   ├── config.py       # Config (KISS)
│   └── exceptions.py   # Исключения
├── entities/           # Сущности (чистая бизнес-логика)
│   └── character.py    # Character (KISS)
├── interfaces/         # Абстракции
│   └── user_interface.py
├── use_cases/          # Сценарии использования
│   └── game.py         # GameUseCase (KISS)
└── ui/                 # Пользовательский интерфейс
    ├── console.py      # Console (KISS)
    └── colors.py       # Цвета (YAGNI)
```

#### Ключевые классы
- **Console** — консольный вывод с цветовой схемой (упрощенный)
- **Game** — основной игровой цикл (координатор)
- **GameUseCase** — логика меню и обработки выбора
- **Character** — бизнес-сущность персонажа

## 🎮 Демонстрация работы

При запуске приложения:
1. Показывается приветственный экран
2. После нажатия Enter появляется главное меню
3. Пользователь выбирает пункт меню вводом цифры 1-5
4. Для неверного ввода показывается ошибка
5. Для выхода из программы выбирается пункт 5
6. **Новая игра** создает персонажа с валидацией
