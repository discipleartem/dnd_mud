# Глава 2: Расы

> Источник: PHB, стр. 17–44. Полный текст — `docs/PHB_ D&D_2023 RUS.pdf` (локально).

## Для игроков

### Выбор расы

Раса задаёт:
- культурное и физическое происхождение;
- **бонусы к характеристикам**;
- скорость, размер, языки;
- **расовые особенности** (тёмное зрение, владение навыками, сопротивления и т.д.).

Некоторые расы имеют **подрасы** (например, горный дварф, высший эльф).

### Девять рас PHB

| Раса | Типичные бонусы (база) |
|------|------------------------|
| Дварф | Телосложение +2 |
| Эльф | Ловкость +2 |
| Полурослик | Ловкость +2 |
| Человек | все характеристики +1 (или вариант: +1 к двум на выбор) |
| Драконорождённый | Сила +2, Харизма +1 |
| Гном | Интеллект +2 |
| Полуэльф | Харизма +2, +1 к двум другим |
| Полуорк | Сила +2, Телосложение +1 |
| Тифлинг | Интеллект +1, Харизма +2 |

Подрасы могут заменять или дополнять базовые бонусы.

### Наследование бонусов подрасы

По PHB подраса может:
- наследовать бонусы базовой расы (`inherit_base_bonuses`);
- добавлять свои бонусы к характеристикам;
- давать **выборные** бонусы (например, variant human: +1 к двум характеристикам на выбор).

### Выборные бонусы (variant human и аналоги)

После распределения базовых значений игрок может выбрать характеристики для дополнительных +1 (или иного значения из `mechanics` особенности расы).

### Ключевые функции

| Функция | Назначение |
|---------|------------|
| `get_race_bonuses(race_id, subrace_id)` | Статические бонусы расы/подрасы |
| `get_choice_ability_bonus_mechanics(...)` | Параметры выборных бонусов |
| `build_bonuses_from_choices(...)` | Сбор словаря из выбора игрока |
| `get_effective_race_bonuses(...)` | Статические + выборные для отображения |
| `load_races(language)` | Список для меню |
| `load_race_full(race_id, language)` | Полное описание для экрана подрасы |

### Формат YAML (фрагмент)

```yaml
races:
  human:
    ability_bonuses: { strength: 1, dexterity: 1, ... }
    subraces:
      variant_human:
        inherit_base_bonuses: false
        features:
          - type: ability_bonus
            mechanics: { count: 2, value: 1, choice: true }
```

UI выборных бонусов: [`ui/menus/stats/stats_choice_bonuses.py`](../../ui/menus/stats/stats_choice_bonuses.py).

### Не реализовано

- Все 9 рас и полный набор подрас PHB
- Механика особенностей в бою (тёмное зрение, сопротивления) — нет game engine
- Дополнительные языки на выбор (только описание в features)
