# Приложения PHB

> Источник: PHB, стр. 290–311. Полный текст — `docs/PHB_ D&D_2023 RUS.pdf` (локально).

## Для игроков

### Приложение А: Состояния

Состояние — игровой эффект с чёткими правилами. Несколько состояний могут действовать одновременно.

| Состояние | Краткий эффект |
|-----------|----------------|
| Бессознательный | Недееспособен; автопровал силовых/ловкостных спасбросков; криты по вам в ближнем бою |
| Испуганный | Помеха, пока источник страха в зоне видимости; не приближаться |
| Невидимый | Невидим; при атаке — преимущество; по вам — помеха |
| Недееспособен | Нет действий, реакций; автопровал силовых/ловкостных спасбросков |
| Оглохший | Автопровал проверок слуха |
| Ограниченный | Скорость 0; помеха спасброскам Ловкости; преимущество атак по вам |
| Окаменевший | Превращение в камень (подробности в PHB) |
| Опутанный | Скорость 0; помеха атакам; преимущество атак по вам |
| Ослеплённый | Автопровал зрения; помеха атакам; преимущество атак по вам |
| Отравленный | Помеха на проверки атаки и характеристик |
| Очарованный | Не атаковать очаровавшего; очаровавший имеет преимущество на соц. взаимодействие |
| Парализованный | Недееспособен; автопровал сил/ловкости; криты по вам в ближнем бою |
| Сбитый с ног | Скорость 0; вставание стоит половину скорости; помеха атак; преимущество ближних атак по вам |
| Ошеломлённый | Недееспособен; не говорит; автопровал сил/ловкости |

### Приложение Б–В: Боги и планы

Справочный лор мультивселенной D&D. Для MUD — опциональный контент модов и приключений.

### Приложение Г: Параметры существ

Таблицы для создания **чудовищ** (КД, HP, атаки, Сл). Используется Мастером / движком при генерации врагов.

### Приложение Д: Литература

Рекомендуемое чтение для вдохновения (не механика).

## Для разработчиков

| Аспект | Значение |
|--------|----------|
| Статус | **Запланировано** |
| YAML | [`database/core/constants.yaml`](../../database/core/constants.yaml) — константы DC/PB |
| Core | Нет `apply_condition()` / `has_condition()` |
| Заметки | Состояния критичны для боя и заклинаний — Phase 2 engine |

### Целевая модель

```python
# Псевдокод
character.conditions: list[str]  # "poisoned", "prone", ...
apply_condition(target, "restrained", source=...)
# Модификаторы атак/спасбросков — по таблице PHB
```
