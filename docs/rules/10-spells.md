# Главы 10–11: Заклинания

> Источник: PHB, стр. 201–289. Полный текст — `docs/PHB_ D&D_2023 RUS.pdf` (локально).

## Для игроков

### Что такое заклинание

Заклинание — магический эффект с правилами **уровня** (заговор = 0), **школы**, **времени накладывания**, **дистанции**, **компонентов** и **длительности**.

### Накладывание

1. Потратить **ячейку** подходящего уровня (заговоры — без ячейки).
2. Выполнить **компоненты**: В (вербальный), С (соматический), М (материальный).
3. Разрешить эффект по описанию.

**Сл спасброска** от заклинания (если есть):

```
Сл = 8 + бонус мастерства + модификатор базовой характеристики заклинателя
```

**Бросок атаки заклинанием** — как обычная атака, урон/эффект по описанию.

### Концентрация

Заклинания с длительностью «концентрация» требуют удержания. При получении урона — спасбросок Телосложения (Сл = 10 или половина урона, что больше). Второе концентрируемое заклинание прекращает первое.

### Ячейки заклинаний

Количество и уровни ячеек — по **таблице класса** и уровню персонажа. Долгий отдых восстанавливает ячейки (кроме особых правил).

### Заговоры (cantrips)

Уровень 0; можно накладывать без ограничения; урон часто растёт на 5/11/17 уровнях персонажа.

### Школы магии

Воплощение, очарование, иллюзия, некромантия, преграждение, преобразование, прорицание, воплощение — категории для описания и сопротивлений.

### Список заклинаний PHB

Глава 11 содержит алфавитный список (~300 заклинаний) с полными описаниями. В MUD не дублируется — см. PDF.
