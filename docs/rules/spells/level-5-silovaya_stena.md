# Силовая Стена

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 5 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | 120 футов |
| Компоненты | В, С, М (щепотка порошка толчёного прозрачного драгоценного камня) |
| Длительность | Концентрация, вплоть до 10 минут |

## Эффект

- Невидимая стена из силового поля появляется в точке, выбранной вами в пределах дистанции.
- Стена появляется, будучи ориентированной в любом направлении: горизонтальном, вертикальном или наклонённом.
- Она может свободно парить, а может опираться на твёрдую поверхность.
