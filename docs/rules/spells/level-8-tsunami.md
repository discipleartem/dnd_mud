# Цунами

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 8 уровень |
| Школа | Вызов (`conjuration`) |
| Время | 1 минута |
| Дистанция | Видимость |
| Компоненты | В, С |
| Длительность | Концентрация, вплоть до 6 раундов |

## Эффект

- В выбранной точке в пределах дистанции появляется стена из воды.
- Вы можете создать стену длиной до 300 футов, высотой 300 футов и толщиной 50 футов.
- Стена существует, пока заклинание активно.
