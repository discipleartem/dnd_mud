# Опутывание

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 1 уровень |
| Школа | Вызов (`conjuration`) |
| Время | 1 действие |
| Дистанция | 90 футов |
| Компоненты | В, С |
| Длительность | Концентрация, вплоть до 1 минуты |

## Эффект

- Из точки в пределах дистанции на площади квадрата с длиной стороны в 20 футов вырастают цепкие травы и плети.
- Пока заклинание активно, эти растения превращают землю в труднопроходимую местность.
- Существа, находящиеся в этой области, когда вы накладываете заклинание, должны преуспеть в спасброске Силы, иначе они станут опутанными растениями до окончания действия заклинания.
