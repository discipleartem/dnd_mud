# Древесный Путь

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 5 уровень |
| Школа | Вызов (`conjuration`) |
| Время | 1 действие |
| Дистанция | На себя |
| Компоненты | В, С |
| Длительность | Концентрация, вплоть до 1 минуты |

## Эффект

- Вы получаете способность входить в дерево и перемещаться через него в другое дерево того же вида, находящееся в пределах 500 футов.
- Оба дерева должны быть живыми, и размер обоих должен быть не меньше вашего.
- Вы должны использовать 5 футов перемещения, чтобы войти в дерево.
