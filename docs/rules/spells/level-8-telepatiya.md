# Телепатия

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 8 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | Без ограничений |
| Компоненты | В, С, М (два переплетённых серебряных кольца) |
| Длительность | 24 часа |

## Эффект

- Вы создаёте телепатическую связь между собой и знакомым согласным существом.
- Это существо может находиться где угодно на том же плане существования, что и вы.
- Заклинание оканчивается, если вы с целью перестанете находиться на одном плане.
