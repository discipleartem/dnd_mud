# Замедление

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 3 уровень |
| Школа | Преобразование (`transmutation`) |
| Время | 1 действие |
| Дистанция | 120 футов |
| Компоненты | В, С, М (капля патоки) |
| Длительность | Концентрация, вплоть до 1 минуты |

## Эффект

- Вы изменяете время для шести существ на ваш выбор, находящихся в кубе с длиной ребра 40 футов в пределах дистанции.
- Все цели должны преуспеть в спасброске Мудрости, иначе они попадут под действие заклинания на время его длительности.
- Скорость попавшего под действие существа уменьшатся вдвое, оно получает штраф −2 к КД и спасброскам Ловкости, и оно не может совершать реакции.
