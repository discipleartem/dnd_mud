# Превращение В Животных

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 8 уровень |
| Школа | Преобразование (`transmutation`) |
| Время | 1 действие |
| Дистанция | 30 футов |
| Компоненты | В, С |
| Длительность | Концентрация, вплоть до 24 часов |

## Эффект

- Ваша магия превращает существ в зверей.
- Выберите любое число согласных существ, которых видите в пределах дистанции.
- Вы меняете облик каждой цели на облик зверя Большого или меньшего размера с показателем опасности не больше 4.
