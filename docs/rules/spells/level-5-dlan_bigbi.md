# Длань Бигби

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 5 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | 120 футов |
| Компоненты | В, С, М (яичная скорлупа и перчатка из змеиной кожи) |
| Длительность | Концентрация, вплоть до 1 минуты |

## Эффект

- Вы создаёте мерцающую руку Большого размера из полупрозрачного силового поля в свободном пространстве, которое видите в пределах дистанции.
- Рука существует, пока активно заклинание, и она перемещается по вашей команде, подражая движениям вашей собственной руки.
- Эта рука является предметом, у которого КД 20, а количество хитов равно максимуму ваших хитов.
