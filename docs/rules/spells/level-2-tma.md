# Тьма

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 2 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | 60 футов |
| Компоненты | В, М (мех летучей мыши и либо капля дёгтя, либо кусочек угля) |
| Длительность | Концентрация, вплоть до 10 минут |

## Эффект

- Из точки, выбранной вами в пределах дистанции, расползается магическая тьма сферой с радиусом 15 футов.
- Тьма огибает углы.
- Существа с тёмным зрением не могут видеть сквозь эту тьму, и немагический свет не может её осветить.
