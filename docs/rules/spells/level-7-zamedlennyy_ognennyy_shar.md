# Замедленный Огненный Шар

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 7 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | 150 футов |
| Компоненты | В, С, М (крошечный шарик из гуано летучей мыши и серы) |
| Длительность | Концентрация, вплоть до 1 минуты |

## Эффект

- Из вашего указательного пальца вылетает луч жёлтого света, концентрирующийся в светящемся шарике в пределах дистанции.
- Когда заклинание оканчивается, либо из-за того, что вы теряете концентрацию, либо когда вы сами это решите, шарик взрывается с гулким рокотом вспышкой пламени, огибающего углы.
- Все существа в сфере с радиусом 20 футов с центром в этой точке должны совершить спасбросок Ловкости.
