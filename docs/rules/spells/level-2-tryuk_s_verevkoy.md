# Трюк С Верёвкой

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 2 уровень |
| Школа | Преобразование (`transmutation`) |
| Время | 1 действие |
| Дистанция | Касание |
| Компоненты | В, С, М (экстракт зерна и петля из пергамента) |
| Длительность | 1 час |

## Эффект

- Вы касаетесь верёвки длиной до 60 футов.
- Один её конец поднимается в воздух, а остальная часть висит перпендикулярно полу.
- На верхнем конце верёвки появляется невидимый вход в межпространство, существующий, пока заклинание активно.
