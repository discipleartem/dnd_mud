# Сглаз

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 1 уровень |
| Школа | Очарование (`enchantment`) |
| Время | 1 бонусное действие |
| Дистанция | 90 футов |
| Компоненты | В, С, М (окаменевший глаз тритона) |
| Длительность | Концентрация, вплоть до 1 часа |

## Эффект

- Вы проклинаете существо, которое видите в пределах дистанции.
- Пока заклинание активно, вы причиняете цели урон некротической энергией 1к6 каждый раз, когда попадаете по ней атакой.
- Кроме того, выберите одну характеристику, когда накладываете заклинание.
