# Вещий Сон

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 5 уровень |
| Школа | Иллюзия (`illusion`) |
| Время | 1 минута |
| Дистанция | Особая |
| Компоненты | В, С, М (горсть песка, капля чернил и перо, выдернутое у спящей птицы) |
| Длительность | 8 часов |

## Эффект

- Это заклинание формирует сон другого существа.
- Выберите целью этого заклинания известное вам существо.
- Цель должна находиться на одном плане существования с вами.
