# Изготовление

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 4 уровень |
| Школа | Преобразование (`transmutation`) |
| Время | 10 минут |
| Дистанция | 120 футов |
| Компоненты | В, С |
| Длительность | Мгновенная |

## Эффект

- Вы превращаете сырьё в готовое изделие из этого материала.
- Например, вы можете создать деревянный мост из нескольких деревьев, верёвку из груды пеньки, или одежду изо льна или шерсти.
- Выберите сырьё, которое видите в пределах дистанции.
