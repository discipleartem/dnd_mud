# Радужные Брызги

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 7 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | На себя (60-футовый конус) |
| Компоненты | В, С |
| Длительность | Мгновенная |

## Эффект

- Восемь разноцветных лучей света вылетают из вашей ладони.
- Каждый луч своего цвета, и у каждого свои особые свойства.
- Все существа в пределах 60-футового конуса должны совершить спасбросок Ловкости.
- Цель получает урон электричеством 10к6 при провале или половину этого урона при успехе.
