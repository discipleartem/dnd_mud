# Зловонное Облако

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 3 уровень |
| Школа | Вызов (`conjuration`) |
| Время | 1 действие |
| Дистанция | 90 футов |
| Компоненты | В, С, М (тухлое яйцо или несколько гнилых листьев капусты) |
| Длительность | Концентрация, вплоть до 1 минуты |

## Эффект

- Вы создаёте сферу с радиусом 20 футов жёлтого тошнотворного газа с центром на точке в пределах дистанции.
- Это облако обходит углы, и его пространство — сильно заслонённая местность.
- Облако держится в воздухе, пока активно заклинание.
