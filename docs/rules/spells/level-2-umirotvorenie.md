# Умиротворение

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 2 уровень |
| Школа | Очарование (`enchantment`) |
| Время | 1 действие |
| Дистанция | 60 футов |
| Компоненты | В, С |
| Длительность | Концентрация, вплоть до 1 минуты |

## Эффект

- Вы пытаетесь подавить сильные эмоции у группы существ.
- Все гуманоиды в сфере с радиусом 20 футов с центром в точке, выбранной в пределах дистанции, должны совершить спасбросок Харизмы; существо может добровольно провалить этот спасбросок, если пожелает.
- Если существо проваливает спасбросок, выберите один из двух описанных ниже эффектов: Вы можете подавить все эффекты, делающие цель очарованной или испуганной.
