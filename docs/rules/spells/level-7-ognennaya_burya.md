# Огненная Буря

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

| Параметр | Значение |
|----------|----------|
| Уровень | 7 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | 150 футов |
| Компоненты | В, С |
| Длительность | Мгновенная |

## Эффект

- В выбранном вами в пределах дистанции пространстве начинается огненная буря.
- Область бури состоит из 1–10 кубов с длиной ребра 10 футов, которые вы можете располагать как вам захочется.
- Каждый куб должен иметь хотя бы одну общую грань с другим кубом.
