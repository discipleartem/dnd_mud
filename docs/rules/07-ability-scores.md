# Глава 7: Использование характеристик

> Источник: PHB, стр. 173–180. Полный текст — `docs/PHB_ D&D_2023 RUS.pdf` (локально).

## Для игроков

### Шесть характеристик

| Характеристика | Отражает |
|----------------|----------|
| Сила | Физическая мощь, атлетизм |
| Ловкость | Проворство, реакция, равновесие |
| Телосложение | Выносливость, здоровье |
| Интеллект | Память, анализ, обучение |
| Мудрость | Внимательность, интуиция |
| Харизма | Обаяние, лидерство, сила воли |

Среднее значение для обычного человека — **10–11** (модификатор +0). Персонажи-искатели обычно 8–15 до расовых бонусов.

### Модификатор

```
модификатор = (значение − 10) // 2
```

Примеры: 8 → −1; 14 → +2; 20 → +5.

### Преимущество и помеха

При **преимуществе** бросают 2к20 и берут **большее**. При **помехе** — **меньшее**.

- Несколько источников преимущества или помехи → всё равно только один дополнительный к20.
- Преимущество и помеха одновременно → **отменяют друг друга** (один к20).

### Бонус мастерства

Добавляется к броску, если персонаж **владеет** соответствующим навыком, спасброском или оружием. На 1 уровне = **+2**. Не складывается дважды с одного источника.

### Проверка характеристики

```
к20 + модификатор характеристики [+ бонус мастерства] [+ прочие]
```

Мастер задаёт **Сл** (сложность). Успех при сумме ≥ Сл.

**Пассивная проверка** (без броска): 10 + все модификаторы. Пример: пассивная Мудрость (Внимательность) = 10 + модификатор Мудрости [+ бонус мастерства, если владеете Внимательностью].

### Навыки

18 навыков привязаны к характеристикам (Атлетика → Сила, Скрытность → Ловкость, …). Владение навыком добавляет бонус мастерства к проверке.

### Спасброски

```
к20 + модификатор характеристики [+ бонус мастерства, если владеете спасброском]
```

Сравнение с Сл эффекта. Класс даёт владение **двумя** спасбросками.

### Соревновательные проверки

Два участника делают проверки одной характеристики; выше результат (или равный при ничьей по правилам Мастера) побеждает.

### Использование характеристик (примеры PHB)

| Характеристика | Примеры действий |
|----------------|------------------|
| Сила | Толкнуть валун, вырваться из захвата |
| Ловкость | Удержать равновесие, акробатический прыжок |
| Телосложение | Удержать дыхание, идти без отдыха |
| Интеллект | Вспомнить знание, расшифровать текст |
| Мудрость | Заметить скрытое, определить настроение |
| Харизма | Убедить, выступить, обмануть |
