# Глава 3: Классы

> Источник: PHB, стр. 45–120. Полный текст — `docs/PHB_ D&D_2023 RUS.pdf` (локально).

## Для игроков

### Что даёт класс

- **Кость хитов** (к6–к12) — для расчёта HP при повышении уровня.
- **Владения**: доспехи, оружие, навыки, инструменты, спасброски.
- **Классовые умения** — от боевых стилей до заклинаний.
- **Подкласс** (архетип) — см. [03-subclasses.md](03-subclasses.md); уровень выбора зависит от класса (1–3).

### Двенадцать классов PHB

| Класс | Кость хитов | Основная характеристика |
|-------|-------------|-------------------------|
| Варвар | к12 | Сила |
| Бард | к8 | Харизма |
| Жрец | к8 | Мудрость |
| Друид | к8 | Мудрость |
| Воин | к10 | Сила или Ловкость |
| Монах | к8 | Ловкость и Мудрость |
| Паладин | к10 | Сила и Харизма |
| Следопыт | к10 | Ловкость и Мудрость |
| Плут | к8 | Ловкость |
| Чародей | к6 | Харизма |
| Колдун | к8 | Харизма |
| Волшебник | к6 | Интеллект |

### Хиты при повышении уровня (PHB)

```
прирост HP = бросок кости хитов (или среднее) + модификатор Телосложения
```

На 1 уровне — максимум кости + модификатор Телосложения (см. [01-character-creation.md](01-character-creation.md)).

### Заклинатели

Бард, жрец, друид, следопыт (частично), чародей, колдун, волшебник — получают **ячейки заклинаний** по уровню. Механика — [10-spells.md](10-spells.md).

## Для разработчиков

| Аспект | Значение |
|--------|----------|
| Статус | Частично: 4 класса в YAML; выбор класса в UI; HP на 1 уровне |
| YAML | [`database/classes/classes.yaml`](../../database/classes/classes.yaml) |
| Core | [`core/classes.py`](../../core/classes.py): `load_classes()`, `get_class_hit_dice()` |
| UI | [`ui/menus/character_flow.py`](../../ui/menus/character_flow.py) — шаг `_select_class` |
| Режимы | Класс не зависит от режима сложности |
| Заметки | `features` и `subclasses` в YAML — справочные; runtime не применяет умения. **Подклассы:** реализуем только перечисленные в YAML (см. [03-subclasses.md](03-subclasses.md) §Политика MUD) |

### Классы в MUD (текущий каталог)

| ID | PHB | hit_dice в YAML |
|----|-----|-----------------|
| `fighter` | Воин | 10 |
| `rogue` | Плут | 8 |
| `cleric` | Жрец | 8 |
| `bard` | Бард | 8 |

Остальные 8 классов PHB — расширение каталога или моды.

### Расчёт HP при создании

```python
# core/character_storage.py
hit_dice = get_class_hit_dice(class_id)
hp = hit_dice + ability_modifier(stats["constitution"])
```

### Формат YAML (фрагмент)

```yaml
classes:
  fighter:
    hit_dice: 10
    prime_ability: strength
    saving_throws: [strength, constitution]
    skill_choices: [athletics, ...]
    skill_choices_count: 2
    features: [...]
    subclasses: [...]
```

### Не реализовано

- Выбор навыков и владений при создании
- Применение `features` в игре
- Выбор подкласса (только из `classes.yaml`) и прогрессия умений — см. [03-subclasses.md](03-subclasses.md); ячейки заклинаний
- Стартовое снаряжение класса
