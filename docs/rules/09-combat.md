# Глава 9: Сражение

> Источник: PHB, стр. 189–198. Полный текст — `docs/PHB_ D&D_2023 RUS.pdf` (локально).

## Для игроков

### Порядок сражения (кратко)

1. **Неожиданность** — кто застигнут врасплох (нет действия/реакции в первом ходу).
2. **Позиции** — где стоят участники.
3. **Инициатива** — проверка Ловкости; порядок ходов на весь бой.
4. **Ходы** — по инициативе до конца раунда.
5. **Новый раунд** — повтор шага 4.

### Ход персонажа

- Перемещение до **скорости** + **одно действие** (порядок на выбор).
- Опционально: **бонусное действие** (если умение разрешает), **реакция** (вне своего хода, 1 за раунд).
- Можно отказаться от движения и действия.

### Основные действия в бою

| Действие | Суть |
|----------|------|
| Атака | Одна или несколько атак (если умение) |
| Засада | Скрытие с проверкой Ловкости (Скрытность) |
| Подготовка | Отложенное действие по триггеру |
| Рывок | Удвоенное перемещение |
| Уклонение | Помеха по атакам по вам до следующего хода |
| Помощь | Преимущество союзнику на следующую проверку/атаку |
| Поиск | Активная Внимательность или Расследование |
| Использование | Взаимодействие с объектом |
| Наложение заклинания | Если время накладывания — «1 действие» |

### Атака

```
бросок атаки = к20 + модификатор характеристики + бонус мастерства [+ прочие]
```

Попадание, если ≥ **КД цели**.

- **Критическое попадание** на натуральной 20: удвоение **костей** урона (не модификаторов).
- **Промах** на натуральной 1 — автоматический промах.

### Урон и лечение

- Урон вычитается из **текущих хитов**.
- При **0 HP** — без сознания; при уроне сверх максимума в отдельных случаях — смерть (правила мгновенной смерти PHB).
- **Стабилизация** и **спасброски от смерти** — каждый ход при 0 HP.
- **Временные хиты** — не складываются с повторным получением; сгорают при долгом отдыхе.

### Укрытие

| Укрытие | Бонус к КД и спасброскам Ловкости |
|---------|-----------------------------------|
| Наполовину | +2 |
| На три четверти | +5 |
| Полное | Нельзя выбрать целью |

## Для разработчиков

| Аспект | Значение |
|--------|----------|
| Статус | **Запланировано** (game engine, Phase 2) |
| YAML | Сценарии боя в `adventures/*.yaml` (будущее) |
| Core | Нет модуля боя; `current_hp` в `Character` — задел |
| Режимы | HardCore = полные правила; Normal = упрощения по дизайну приключения |
| Заметки | `core/dice.py` — только `roll()`; нет `attack_roll()` / `damage()` |

### Минимальный API (целевой)

| Функция | Назначение |
|---------|------------|
| `roll_initiative(dex_mod)` | Инициатива |
| `attack_roll(...)` | к20 + модификаторы, крит на 20 |
| `apply_damage(character, amount)` | HP, unconscious, death saves |
| `healing(...)` | Лечение, temp HP |
