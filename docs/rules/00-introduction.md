# Введение

> Источник: PHB, стр. 5–9. Полный текст — `docs/PHB_ D&D_2023 RUS.pdf` (локально).

## Для игроков

### Что такое D&D

D&D — настольная ролевая игра о приключениях в фэнтезийном мире. Участники создают **персонажей** (искателей приключений) и описывают их действия; исходы неопределённых ситуаций решают **броски кубиков**.

### Роли за столом

| Роль | Кто | Задача |
|------|-----|--------|
| Игрок | Участник | Один персонаж: решения, реплики, броски |
| Мастер (DM) | Один из участников | Мир, NPC, чудовища, последствия, Сл проверок |

В **dnd_mud** роль Мастера выполняет движок и YAML-сценарии; игрок выбирает варианты из нумерованных списков.

### Основной цикл игры

1. Мастер (или игра) описывает ситуацию.
2. Игрок объявляет действие персонажа.
3. При необходимости — бросок к20 + модификаторы.
4. Сравнение с целевым числом (Сл) или КД цели.
5. Описание результата.

### Игральные кости

Обозначение: **кN** — куб с N гранями (к4, к6, к8, к10, к12, к20, к100).

| Обозначение | Значение |
|-------------|----------|
| 1к8 | один к8 |
| 2к6 | два к6, сумма |
| 1к8 + 3 | к8 + модификатор 3 |

**К20** — главный куб для проверок, спасбросков и атак.

### Три типа бросков на к20

| Тип | Формула (упрощённо) |
|-----|---------------------|
| Проверка характеристики | к20 + модификатор характеристики [+ бонус мастерства] |
| Спасбросок | к20 + модификатор характеристики [+ бонус мастерства, если владеете] |
| Бросок атаки | к20 + модификатор характеристики [+ бонус мастерства] [+ прочие] |

Успех — если сумма **≥** целевого числа (Сл или КД).

## Для разработчиков

| Аспект | Значение |
|--------|----------|
| Статус | Частично: только генерация костей |
| YAML | — |
| Core | [`core/dice.py`](../../core/dice.py): `roll()`, `roll_ability_score()`, `ability_modifier()` |
| Режимы | Одинаковые формулы кубов; режим влияет на контент и строгость engine (Phase 2) |
| Заметки | Проверки, спасброски, атаки — не реализованы; см. [07-ability-scores.md](07-ability-scores.md), [09-combat.md](09-combat.md) |

### Реализованные функции кубов

```python
roll(count=1, sides=20, modifier=0)      # общий бросок
roll_ability_score()                        # 4к6, отбросить наименьший
ability_modifier(score)                     # (score - 10) // 2
```
