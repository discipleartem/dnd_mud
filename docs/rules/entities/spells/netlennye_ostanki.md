# Нетленные Останки

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

<!-- phb:auto:parameters -->
| Параметр | Значение |
|----------|----------|
| Уровень | 2 уровень |
| Школа | Некромантия (ритуал) (`некромантия (ритуал)`) |
| Время | 1 действие |
| Дистанция | Касание |
| Компоненты | В, С, М (щепотка соли и по одной медной монетке на каждый глаз трупа, которые должны оставаться там всё время, пока активно заклинание) |
| Длительность | 10 дней Вы касаетесь трупа или других останков. Пока заклинание активно, цель защищена от разложения и не может стать нежитью. Это заклинание также увеличивает ограничение по времени, в течение которого труп можно воскресить, так как дни, проведённые под воздействием этого заклинания, не учитываются при подсчёте времени со дня смерти для таких заклинаний как оживление. |
