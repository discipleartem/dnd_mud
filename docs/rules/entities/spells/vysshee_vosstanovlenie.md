# Высшее Восстановление

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

<!-- phb:auto:parameters -->
| Параметр | Значение |
|----------|----------|
| Уровень | 5 уровень |
| Школа | Ограждение (`abjuration`) |
| Время | 1 действие |
| Дистанция | Касание |
| Компоненты | В, С, М (бриллиантовая пыль, стоящая как минимум 100 зм, расходуемая заклинанием) |
| Длительность | Мгновенная |
<!-- /phb:auto:parameters -->

## Эффект

<!-- phb:auto:effect -->
- Вы наделяете существо, которого касаетесь, позитивной энергией, отменяющей ослабляющие эффекты.
- Вы можете уменьшить уровень истощения цели на 1, или окончить на ней один из следующих эффектов: • • • • Один эффект, делающий цель очарованной или окаменевшей Одно проклятье, включая настройку на проклятый магический предмет Любое уменьшение одной из характеристик цели Один эффект, уменьшающий максимум хитов цели
<!-- /phb:auto:effect -->
