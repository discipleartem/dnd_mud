# Дневной Свет

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

<!-- phb:auto:parameters -->
| Параметр | Значение |
|----------|----------|
| Уровень | 3 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | 60 футов |
| Компоненты | В, С |
| Длительность | 1 час |
<!-- /phb:auto:parameters -->

## Эффект

<!-- phb:auto:effect -->
- Из точки, выбранной вами в пределах дистанции, распространяется сфера света с радиусом 60 футов.
- Сфера излучает яркий свет и тусклый свет в пределах ещё 60 футов.
- Если выбранная вами точка находится на предмете, который вы держите, несёте или носите, свет исходит из предмета и перемещается вместе с ним.
- Если полностью накрыть источник света непрозрачным предметом, например, чашей или шлемом, свет будет заблокирован.
- Если часть зоны этого заклинания накрывает зону тьмы, созданной заклинанием с уровнем не выше 3, заклинание, создавшее тьму, рассеивается.
<!-- /phb:auto:effect -->
