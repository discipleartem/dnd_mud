# Изгнание

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

<!-- phb:auto:parameters -->
| Параметр | Значение |
|----------|----------|
| Уровень | 4 уровень |
| Школа | Ограждение (`abjuration`) |
| Время | 1 действие |
| Дистанция | 60 футов |
| Компоненты | В, С, М (неприятный для цели предмет) |
| Длительность | Концентрация, вплоть до 1 минуты |
<!-- /phb:auto:parameters -->

## Эффект

<!-- phb:auto:effect -->
- Вы пытаетесь послать одно существо, которое видите в пределах дистанции, на другой план существования.
- Цель должна преуспеть в спасброске Харизмы, иначе будет изгнана.
<!-- /phb:auto:effect -->
