# Снятие Проклятья

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

<!-- phb:auto:parameters -->
| Параметр | Значение |
|----------|----------|
| Уровень | 3 уровень |
| Школа | Ограждение (`abjuration`) |
| Время | 1 действие |
| Дистанция | Касание |
| Компоненты | В, С |
| Длительность | Мгновенная |
<!-- /phb:auto:parameters -->

## Эффект

<!-- phb:auto:effect -->
- После вашего касания все проклятья, действующие на одно существо или предмет, оканчиваются.
- Если предмет был проклятым магическим предметом, проклятье остаётся, но заклинание разрывает настройку владельца на него, поэтому его можно снять или выбросить.
<!-- /phb:auto:effect -->
