# Послание

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

<!-- phb:auto:parameters -->
| Параметр | Значение |
|----------|----------|
| Уровень | 3 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | Неограниченная |
| Компоненты | В, С, М (кусок тонкой медной проволоки) |
| Длительность | 1 раунд |
<!-- /phb:auto:parameters -->

## Эффект

<!-- phb:auto:effect -->
- Вы посылаете короткое сообщение длиной не больше 25 слов существу, известному вам.
- Это существо слышит послание в сознании, распознаёт вас в качестве отправителя, если оно вас знает, и может тут же ответить.
- Это заклинание позволяет 262
<!-- /phb:auto:effect -->
