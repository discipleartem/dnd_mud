# Лабиринт

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

<!-- phb:auto:parameters -->
| Параметр | Значение |
|----------|----------|
| Уровень | 8 уровень |
| Школа | Вызов (`вызов`) |
| Время | 1 действие |
| Дистанция | 60 футов |
| Компоненты | В, С |
| Длительность | Концентрация, вплоть до 10 минут |
<!-- /phb:auto:parameters -->

## Эффект

<!-- phb:auto:effect -->
- Вы изгоняете существо, видимое в пределах дистанции, в похожий на лабиринт демиплан.
- Цель остаётся там, пока заклинание активно, или пока она не найдёт выход.
- Цель может действием пытаться найти выход.
- Для этого она совершает проверку Интеллекта со 234
<!-- /phb:auto:effect -->
