# Множественное Лечение Ран

> Источник: PHB, гл. 11. Пересказ правил, не дословная копия PHB.

## Параметры

<!-- phb:auto:parameters -->
| Параметр | Значение |
|----------|----------|
| Уровень | 5 уровень |
| Школа | Воплощение (`evocation`) |
| Время | 1 действие |
| Дистанция | 60 футов |
| Компоненты | В, С |
| Длительность | Мгновенная |
<!-- /phb:auto:parameters -->

## Эффект

<!-- phb:auto:effect -->
- 240 Волна целительной энергии исходит из точки, выбранной вами в пределах дистанции.
- Выберите до шести существ в сфере с радиусом 30 футов с центром на этой точке.
- Все цели восстанавливают хиты в количестве, равном 3к8 + модификатор вашей базовой характеристики.
- Это заклинание не оказывает никакого эффекта на нежить и конструктов.
- это заклинание, используя ячейку 6 уровня или выше, восстанавливается дополнительно 1к8 хитов за каждый уровень ячейки выше пятого.
<!-- /phb:auto:effect -->
