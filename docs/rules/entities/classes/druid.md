# Друид

> Источник: PHB, стр. 64–69. Только механика.

## Правила (PHB)

<!-- phb:auto:summary -->
### Хиты и основа

- Кость хитов: 1к8; подкласс (круг) с 2 уровня.
- Заклинания: полный прогресс; базовая характеристика — Мудрость.

### Владения

- Доспехи: лёгкие, средние (неметалл.), щиты (неметалл.).
- Оружие: булавы, кинжалы, дротики, дубинки, метательные копья, посохи, скимитары, серпы, пращи.
- Инструменты: набор травника.
- Спасброски: Интеллект, Мудрость.
- Навыки: 2 из Внимательность, Выживание, Магия, Медицина, Обращение с животными, Природа, Проницательность, Религия.

### Ключевые умения

- **1 ур.:** Друидический язык; Заговоры.
- **2 ур.:** Дикий облик; Круг друида.
- **4 ур.:** Улучшение дикого облика (к1/2 ур. друида).
- **18 ур.:** Заклинания в диком облике.
- **20 ур.:** Архидруид (безлимитный дикий облик).

### Круги (подклассы)

- **Круг земли:** Дополнительные заговоры/подготовленные заклинания по биому.
- **Круг луны:** Боевой дикий облик; дополнительные облики.
