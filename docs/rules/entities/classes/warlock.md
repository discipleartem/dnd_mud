# Колдун

> Источник: PHB, стр. 105–111. Только механика.

## Правила (PHB)

<!-- phb:auto:summary -->
### Хиты и основа

- Кость хитов: 1к8; покровитель и договор с 1/3 уровня.
- Заклинания: магия договора (ячейки восстанавливаются на коротком отдыхе); Харизма.

### Владения

- Доспехи: лёгкие.
- Оружие: простое.
- Спасброски: Мудрость, Харизма.
- Навыки: 2 из списка.

### Ключевые умения

- **1 ур.:** Покровитель; Магия договора.
- **2 ур.:** Таинственные воззвания.
- **3 ур.:** Договор (цепи / клинок / книга).
- **11 ур.:** Таинственный арканум (6 ур.).
- **17 ур.:** Таинственный арканум (7–9 ур.).
- **20 ур.:** Мастер воззваний.

### Договоры

- **Цепи:** Фамильяр.
- **Клинок:** Призыв оружия договора.
- **Книга:** Книга теней.
