# Следопыт

> Источник: PHB, стр. 89–93. Только механика.

## Правила (PHB)

<!-- phb:auto:summary -->
### Хиты и основа

- Кость хитов: 1к10; архетип с 3 уровня.
- Заклинания: полузаклинатель; базовая характеристика — Мудрость.

### Владения

- Доспехи: лёгкие, средние, щиты.
- Оружие: простое, воинское.
- Спасброски: Сила, Ловкость.
- Навыки: 3 из списка (включая любимую местность).

### Ключевые умения

- **1 ур.:** Любимый враг; Исследователь природы.
- **2 ур.:** Боевой стиль; Использование заклинаний.
- **3 ур.:** Архетип следопыта.
- **5 ур.:** Дополнительная атака.
- **8 ур.:** Улучшение местности; Скрытность в любимой местности.
- **10 ур.:** Укрытие в любимой местности.
- **14 ур.:** Маскировка в любимой местности.
- **18 ур.:** Дикие чувства.

### Архетипы

- **Охотник:** Добыча охотника; Оборонительная тактика; Мультиатака.
- **Повелитель зверей:** Спутник зверя; Исключительный спутник.
