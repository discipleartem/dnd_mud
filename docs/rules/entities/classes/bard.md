# Бард

> Источник: PHB, стр. 51–55. Только механика.

## Правила (PHB)

<!-- phb:auto:summary -->
### Хиты и основа

- Кость хитов: 1к8; коллегия бардов с 3 уровня.
- Заклинания: полный прогресс; базовая характеристика — Харизма.

### Владения

- Доспехи: лёгкие.
- Оружие: простое, ручной арбалет, длинный меч, рапира, короткий меч.
- Инструменты: 3 музыкальных на выбор.
- Спасброски: Ловкость, Харизма.
- Навыки: 3 любых на выбор.

### Ключевые умения

- **1 ур.:** Использование заклинаний; Вдохновение барда (кость к6, бонусным действием союзнику в 60 фт.; использований = мод. Харизмы, мин. 1).
- **2 ур.:** Мастер на все руки; Песнь отдыха (доп. исцеление после короткого отдыха).
- **3 ур.:** Компетентность (2 навыка); Коллегия бардов.
- **5 ур.:** Источник вдохновения (восстановление после короткого отдыха).
- **6 ур.:** Контрочарование.
- **10 ур.:** Компетентность (ещё 2); Тайны магии (2 заклинания любого класса).
- **14 ур.:** Тайны магии (ещё 2).
- **18 ур.:** Тайны магии (ещё 2).
- **20 ур.:** Превосходное вдохновение (кость вдохновения при инициативе).

### Коллегии

- **Доблести:** Средние доспехи, щиты, воинское оружие; Боевое вдохновение; Дополнительная атака (6 ур.).
- **Знаний:** 3 доп. навыка; Острое словцо; Дополнительные тайны магии (6 ур.).
