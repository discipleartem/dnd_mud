# Чародей

> Источник: PHB, стр. 99–104. Только механика.

## Правила (PHB)

<!-- phb:auto:summary -->
### Хиты и основа

- Кость хитов: 1к6; происхождение чародея с 1 уровня.
- Заклинания: полный прогресс; базовая характеристика — Харизма.

### Владения

- Доспехи: нет.
- Оружие: дротики, кинжалы, пращи, посохи, лёгкие арбалеты.
- Спасброски: Телосложение, Харизма.
- Навыки: 2 из списка.

### Ключевые умения

- **1 ур.:** Происхождение (драконья кровь / дикая магия).
- **2 ур.:** Источник магии (очки чародейства).
- **3 ур.:** Метамагия.
- **4/10/17 ур.:** Дополнительные известные заклинания.
- **20 ур.:** Восстановление чар (4 очка чародейства после короткого отдыха).

### Происхождения

- **Драконья кровь:** Сопротивление стихии; дыхание дракона.
- **Дикая магия:** Всплески дикой магии; Управление судьбой.
