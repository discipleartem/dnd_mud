### Метод генерации 4d6 drop lowest
```python
def generate_ability_score() -> int:
    """Генерация характеристики методом 4d6 drop lowest"""
    rolls = [random.randint(1, 6) for _ in range(4)]
    rolls.remove(min(rolls))  # Удаляем наименьший бросок
    return sum(rolls)
```

### Расчет модификаторов
```python
def calculate_modifier(score: int) -> int:
    """Расчет модификатора характеристики"""
    return (score - 10) // 2
```
