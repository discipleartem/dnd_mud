# Подзадача 2: Выбор расы/подрасы

## Цель
Предоставить выбор расы и подрасы из доступных в data/races.yaml

## Требования
- Загрузка данных о расах из YAML файла
- Отображение доступных рас с описаниями
- Выбор подрасы (если применимо)
- Применение расовых бонусов к характеристикам
- Установка расовых особенностей (traits)

## Критерии выполнения
- Корректная загрузка данных из YAML
- Интерактивный выбор расы/подрасы
- Применение модификаторов к характеристикам
- Сохранение расовых данных в профиле

## Техническая реализация

### Компоненты
- **RaceDataLoader** - загрузка из YAML
- **RaceSelector** - интерфейс выбора
- **RaceBonusApplier** - применение бонусов

### Структура данных YAML
```yaml
races:
  human:
    name: "Человек"
    description: "..."
    ability_bonuses:
      strength: 1
      dexterity: 1
      constitution: 1
      intelligence: 1
      wisdom: 1
      charisma: 1
    traits:
      - "Дополнительная способность"
    subraces:
      standard_human:
        name: "Обычный человек"
        # ... дополнительные бонусы
```

1. Загрузка данных из races.yaml
2. Отображение списка рас с описаниями
3. Выбор расы пользователем
4. Проверка наличия подрас
5. Выбор подрасы (если есть)
6. Применение бонусов к характеристикам
7. Сохранение расовых данных
