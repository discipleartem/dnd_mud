# Подзадача 7: Выбор языков

## Цель
Определение языков, которыми владеет персонаж

## Требования
- Учет языков от расы/класса/предыстории
- Возможность выбора дополнительных языков
- Ограничение по количеству языков
- Стандартные языки D&D

## Стандартные языки D&D
### Общие языки
- **Common** (Общий)
- **Dwarvish** (Дварфийский)
- **Elvish** (Эльфийский)
- **Giant** (Гигантский)
- **Gnomish** (Гномий)
- **Goblin** (Гоблинский)
- **Halfling** (Полурослийский)
- **Orc** (Орочий)

### Экзотические языки
- **Abyssal** (Бездонный)
- **Celestial** (Небесный)
- **Draconic** (Драконий)
- **Deep Speech** (Глубинная речь)
- **Infernal** (Инфернальный)
- **Primordial** (Первородный)
- **Sylvan** (Лесной)
- **Undercommon** (Подземный)

## Критерии выполнения
- Корректный учет базовых языков
- Интерактивный выбор дополнительных
- Проверка лимитов на количество языков

## Техническая реализация

### Компоненты
- **LanguageManager** - управление языками
- **LanguageSelector** - интерфейс выбора
- **LanguageData** - справочник языков

### Структура данных
```python
@dataclass
class Language:
    name: str
    name_ru: str
    type: str  # "common" или "exotic"
    script: Optional[str] = None
    typical_speakers: List[str] = field(default_factory=list)
```

### Алгоритм
1. Сбор базовых языков от расы/класса/предыстории
2. Определение количества доступных дополнительных языков
3. Отображение доступных языков
4. Интерактивный выбор дополнительных языков
5. Проверка лимитов
6. Формирование финального списка языков
7. Сохранение данных

### Правила выбора
- Базовые языки от расы всегда доступны
- Некоторые классы дают дополнительные языки (например, Bard)
- Предыстория может добавлять 1-2 языка
- Интеллект может давать дополнительные языки (бонус от характеристики)
