# Подзадача 1: Ввод имени

## Цель
Получить от игрока имя персонажа

## Требования
- Валидация формата имени (длина, символы)
- Возможность повторного ввода при ошибке
- Сохранение имени в профиле персонажа

## Критерии выполнения
- ✅ Обработка ошибок ввода
- ✅ Сохранение в character repository

## Техническая реализация

### Компоненты
- ✅ **NameValidator** - валидация формата имени
- ✅ **CharacterRepository** - проверка уникальности
- ✅ **ConsoleAdapter** - взаимодействие с пользователем

### Правила валидации
- ✅ Минимальная длина: 3 символа
- ✅ Максимальная длина: 16 символов
- ✅ Разрешенные символы: буквы, дефисы
- ✅ Запрещены: цифры, спецсимволы (кроме дефиса)

### Алгоритм
1. ✅ Запрос имени у пользователя
2. ✅ Валидация формата
3. ✅ При ошибке - повторный запрос с объяснением
4. ✅ Сохранение в профиль персонажа
