## Структура docstring
```python
def roll_dice(dice_notation: str, advantage: bool = False) -> DiceResult:
    """
    Бросок кубика по нотации D&D 5e.
    
    Поддерживает стандартную нотацию кубиков: "1d20", "2d6+3", "3d8-2".
    При advantage бросается дважды и выбирается лучший результат.
    
    Args:
        dice_notation: Нотация кубика (например, "1d20+5")
        advantage: Если True, бросается дважды и выбирается максимум
    
    Returns:
        DiceResult: Результат броска с деталями (сумма, отдельные броски, модификатор)
    
    Raises:
        ValueError: Если нотация некорректна
    
    Examples:
        >>> result = roll_dice("1d20+5")
        >>> result.total  # Сумма броска с модификатором
        
        >>> result = roll_dice("1d20", advantage=True)
        >>> result.total  # Максимум из двух бросков
    
    Note:
        Соответствует правилам D&D 5e для бросков кубиков.
    """
```
