# Этап 3: Расширенное создание персонажа

**Срок:** 4-6 недель  
**Приоритет:** Высокий

## Задачи этапа

### 3.1 Выбор расы персонажа [HIGH]
- **Цель:** Добавить систему рас с механиками D&D 5e
- **Требования:**
  - Создать `data/races.yaml` с данными рас
  - Реализовать сущность Race в `entities/race.py`
  - Добавить механики расовых бонусов
  - Интегрировать в процесс создания персонажа
- **Файлы для создания/изменения:**
  - `data/races.yaml`
  - `entities/race.py`
  - `use_cases/character_creation.py`

### 3.2 Выбор класса персонажа [HIGH]
- **Цель:** Добавить классы с уникальными способностями
- **Требования:**
  - Создать `data/classes.yaml` с данными классов
  - Реализовать сущность Class в `entities/class.py`
  - Добавить базовые способности классов
  - Интегрировать в процесс создания персонажа
- **Файлы для создания/изменения:**
  - `data/classes.yaml`
  - `entities/class.py`
  - `use_cases/character_creation.py`

### 3.3 Генерация характеристик [HIGH]
- **Цель:** Добавить систему характеристик (STR, DEX, CON, INT, WIS, CHA)
- **Требования:**
  - Реализовать сущность Abilities в `entities/abilities.py`
  - Добавить механику генерации (4d6 drop lowest)
  - Рассчитывать модификаторы характеристик
  - Применять расовые бонусы
- **Файлы для создания/изменения:**
  - `entities/abilities.py`
  - `entities/character.py` (обновление)
  - `use_cases/character_creation.py`

### 3.4 Выбор предыстории персонажа [HIGH]

- **Файлы для создания/изменения:**
  - `data/backgrounds.yaml`

  - `use_cases/character_creation.py`

### 3.5 Выбор навыков [HIGH]
