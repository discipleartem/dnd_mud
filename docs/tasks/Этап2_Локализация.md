# Этап 2: Локализация (i18n)

**Срок:** 2-3 недели  
**Приоритет:** Высокий

## Архитектурный принцип

**Полная локализация всего контента через YAML:**
- Все текстовые строки вынесены в YAML файлы
- Поддержка множественных языков
- Автоматическое определение языка системы
- API для динамической локализации
- Падежи и формы для русского языка

## Задачи этапа

### 2.1 Система локализации [HIGH]
- **Цель:** Создать базовую систему интернационализации
- **Требования:**
  - Реализовать сущность I18n в `entities/i18n.py`
  - Создать API для локализации строк
  - Добавить систему определения языка системы
  - Реализовать загрузку языковых файлов
  - Создать систему кэширования переводов
- **Файлы для создания/изменения:**
  - `entities/i18n.py`
  - `interfaces/i18n_api.py`
  - `use_cases/i18n_manager.py`
  - `data/i18n/` (структура языковых файлов)

### 2.2 Языковые файлы интерфейса [HIGH]
- **Цель:** Вынести все строки интерфейса в YAML
- **Требования:**
  - Создать `data/i18n/ru.yaml` для русского языка
  - Создать `data/i18n/en.yaml` для английского языка
  - Локализовать главное меню и навигацию
  - Локализовать все UI элементы
  - Добавить поддержку горячих клавиш
- **Файлы для создания/изменения:**
  - `data/i18n/ru.yaml`
  - `data/i18n/en.yaml`
  - `src/ui/console.py` (интеграция i18n)
  - `src/interfaces/user_interface.py` (обновление)

### 2.3 Локализация персонажей [HIGH]

- **Требования:**

## Структура языковых файлов

```
data/i18n/
├── ru.yaml                 # Основной русский язык
├── en.yaml                 # Английский язык
├── characters.yaml         # Локализация персонажей
├── mechanics.yaml          # Локализация механик
├── adventures.yaml         # Локализация приключений
├── abilities.yaml          # Локализация способностей
├── spells.yaml            # Локализация заклинаний
├── feats.yaml             # Локализация черт
├── dialogues.yaml         # Локализация диалогов
├── npc.yaml               # Локализация NPC
├── grammar_rules.yaml     # Правила грамматики
└── templates/             # Шаблоны локализации
    ├── character_sheets/
    ├── ui_elements/
    └── adventure_descriptions/
```

## Пример YAML структуры

```yaml
# ru.yaml
ui:
  main_menu:
    title: "D&D Text MUD"
    new_game: "Новая игра"
    load_game: "Загрузить игру"
    settings: "Настройки"
    exit: "Выход"

character:
  races:
    human: "Человек"
    elf: "Эльф"
    dwarf: "Дварф"
  
  classes:
    fighter: "Воин"
    wizard: "Волшебник"
    rogue: "Плут"

grammar:
  numerals:
    one: "один"
    few: "два"
    many: "пять"
```
