# Этап 1: Приветственное окно и базовая архитектура

## Обзор этапа
Создание фундаментальной архитектуры проекта согласно Clean Architecture и базового приветственного интерфейса.

## Цели этапа
- Настроить структуру проекта согласно принципам Clean Architecture
- Создать базовые интерфейсы и системы конфигурации
- Реализовать приветственный экран с переходом к главному меню

## Задачи этапа

### 1.1 Архитектура проекта
- [x] **TASK-1.1.1**: Создать структуру согласно Clean Architecture
  - Папки: entities, use_cases, interfaces, adapters, dtos, console
  - Базовые интерфейсы и абстрактные классы
  - Настройка mypy для type checking

- [x] **TASK-1.1.2**: Создать систему конфигурации
  - Конфигурационный файл settings.yaml
  - Класс Settings для управления настройками
  - Поддержка языков (ru/en)

- [x] **TASK-1.1.3**: Настроить логирование
  - Базовый логгер для отладки
  - Уровни логирования
  - Запись в файл при ошибках

### 1.2 Приветственное окно
- [x] **TASK-1.2.1**: Создать адаптер приветственного окна
  - Класс WelcomeScreenAdapter
  - Цветной логотип с colorama
  - Краткое описание игры

- [x] **TASK-1.2.2**: Интеграция с главным меню
  - Переход к главному меню
  - Обработка ввода пользователя

## Результат этапа
Полностью настроенная архитектура проекта с работающим приветственным экраном и готовностью к реализации главного меню.
