# Phase 3: Создание персонажа

## Обзор

Система создания персонажей для D&D Text MUD, следующая правилам D&D 5e.

## Задачи

### 3.1 Базовая система создания персонажа
- Выбор расы и класса
- Распределение характеристик
- Выбор навыков и умений

### 3.2 Валидация персонажа
- Проверка правил D&D
- Баланс характеристик
- Корректность данных

### 3.3 Сохранение персонажа
- Интеграция с системой сохранений
- Экспорт персонажа
- Импорт персонажа
