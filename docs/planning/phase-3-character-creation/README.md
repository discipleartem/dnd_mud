# Этап 3: Создание персонажа (MVP приоритет)

## Обзор этапа
Полноценная система создания персонажей Dungeons & Dragons 5e со всеми механиками из Player's Handbook.

## Цели этапа
- Реализовать полный цикл создания персонажа D&D 5e
- Поддержать все расы, классы, предыстории из Книги Игрока
- Создать интуитивный пошаговый интерфейс

## Задачи этапа

### 3.1 Базовая система создания персонажа
- [ ] **TASK-3.1.1**: Создать сервис создания персонажа
  - CharacterCreationService (основная логика)
  - CharacterDTO для данных персонажа
  - Валидация каждого этапа

- [ ] **TASK-3.1.2**: Адаптер создания персонажа
  - CharacterCreationAdapter (UI слой)

  - Сохранение прогресса создания

### 3.2 Имя персонажа

- [ ] **TASK-3.2.2**: UI ввода имени
