# Этап 6: Основные игровые системы

## Обзор этапа
Реализация ключевых игровых механик D&D 5e: боевая система, уровни, торговля, репутация.

## Цели этапа
- Создать минимально необходимую боевую систему
- Реализовать систему уровней и прокачки (до 10 уровня)
- Добавить торговлю и репутацию

## Задачи этапа

### 6.1 Боевая система (минимальная)
- [ ] **TASK-6.1.1**: Движок боя
  - CombatEngine
  - Инициатива, атаки, урон
  - Пошаговая логика

- [ ] **TASK-6.1.2**: UI боя
  - CombatUIAdapter
  - Отображение поля боя
  - Команды действий

### 6.2 Система уровней и прокачки
- [ ] **TASK-6.2.1**: Сервис уровней
  - LevelService
  - Опыт и повышение уровней (до 10)
  - Увеличение характеристик

- [ ] **TASK-6.2.2**: UI прокачки
  - LevelUpUIAdapter
  - Выбор новых способностей
  - Увеличение характеристик

### 6.3 Торговая система
- [ ] **TASK-6.3.1**: Сервис торговли
  - TradeService
  - Покупка/продажа
  - Цены и деньги

- [ ] **TASK-6.3.2**: UI торговли
  - TradeUIAdapter
  - Меню торговца
  - Инвентарь игрока

### 6.4 Система репутации
- [ ] **TASK-6.4.1**: Сервис репутации

  - Влияние на мировоззрение

- [ ] **TASK-6.4.2**: UI репутации

  - Команды репутации

## Результат этапа
Базовые игровые механики D&D 5e, готовые для использования в приключениях и дальнейшего расширения.
