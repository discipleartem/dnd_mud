# PRD: D&D Text MUD - Консольная текстовая игра Dungeons & Dragons 5e

## Краткое описание
Создание пошаговой консольной RPG игры по мотивам Dungeons & Dragons 5e с системой создания персонажей, модулями приключений и подготовкой к будущему многопользовательскому MUD режиму.

## Обзор проекта

### Цель проекта
Разработать однопользовательскую консольную текстовую RPG с полной механикой D&D 5e, которая будет расширяться до многопользовательского MUD.

### Технологический стек
- Python 3.12+
- pytest, black, ruff, mypy
- PyYAML (для игровых данных)
- colorama (для цветного вывода)
- JSON (для сохранений)

### Архитектурные принципы
- Clean Architecture с разделением на слои
- YAML файлы как база данных игровых механик
- JSON для сохранения состояния персонажей и игры
- Модульная система для расширений

## Этап 2: Главное меню

### Функции меню:
1. **Новая игра** - запуск приключения
2. **Создание персонажа** - полный генератор персонажей
3. **Загрузка игры** - выбор сохранения из JSON файлов
4. **Настройки** - конфигурация игры (язык, hardcore режим)
5. **Моды** - управление модификациями
6. **Выход** - завершение игры

### Задачи:
1. **Разработка системы меню**
   - Создание базового класса меню
   - Навигация между пунктами
   - Цветовая схема для разных элементов

2. **Система настроек**
   - Переключение языка (русский/английский)
   - Hardcore режим (ограничения инвентаря)
   - Сохранение настроек в конфиг

3. **Система сохранений**
   - Чтение/запись JSON файлов
   - Валидация сохранений
   - Список доступных сохранений

## Этап 3: Создание персонажа (MVP приоритет)

### 3.1 Выбор имени
- Валидация: не пустое, до 20 символов
- Разрешенные символы: буквы, дефис, пробелы
- Обрезка пробелов в начале/конце

### 3.2 Выбор расы и подрасы
- Человек (базовый и вариант)
- Эльф и подрасы
- Дворф и подрасы
- Полуорк
- Применение расовых бонусов к характеристикам

### 3.3 Генерация характеристик
- **Стандартный метод**: выбор из [15, 14, 13, 12, 10, 8]
- **Point Buy**: 27 очков, стоимость по таблице
- **Случайная генерация**: 4d6 drop lowest
- Применение расовых бонусов

### 3.4 Выбор класса
- Воин (с архетипами)
- Варвар (с архетипами)
- Бард (с архетипами)
- Плут (с архетипами)
- Жрец (с архетипами)

### 3.5 Выбор предыстории
- Из Книги Игрока
- Применение бонусов к навыкам

### 3.6 Выбор навыков
- Расовые бонусы
- Классовые навыки (выбор из списка)
- Бонусы от предыстории

### 3.7 Выбор черт
- Расовые черты
- Классовые черты (если есть)
- Черты от предыстории

### 3.8 Выбор языков
- Расовые языки
- Дополнительные языки от класса/предыстории

### 3.9 Выбор заклинаний/заговоров
- Для заклинателей: выбор доступных заклинаний
- Применение бонусов от расы/класса

### 3.10 Выбор снаряжения
- Стартовое снаряжение класса
- Дополнительное от предыстории
- Альтернатива: начальное богатство (данные из таблицы)

### 3.11 Дополнительная информация
- Возраст (зависит от расы) - обязательно
- Мировоззрение - обязательно
- Рост/вес (зависит от расы) - опционально
- Внешность, черты характера, идеалы - опционально
