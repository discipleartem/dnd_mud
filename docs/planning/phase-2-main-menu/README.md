# Этап 2: Главное меню

## Обзор этапа
Разработка полнофункционального главного меню с навигацией по всем основным модулям игры.

## Цели этапа
- Создать гибкую систему меню с цветовой схемой
- Реализовать управление настройками игры
- Настроить систему сохранений и загрузки игр

## Задачи этапа

### 2.1 Базовая система меню
- [x] **TASK-2.1.1**: Создать базовый класс меню
  - AbstractMenu с навигацией
  - MenuRenderer для отрисовки
  - Цветовая схема элементов меню

- [x] **TASK-2.1.2**: Реализовать главное меню
  - 6 пунктов меню (Новая игра, Создать персонажа, Загрузить игру, Настройки, Languages, Выход)
  - Навигация стрелками и цифрами
  - Интеграция с существующим приветственным экраном
  - Поддержка горячих клавиш

### 2.2 Настройки игры
- [x] **TASK-2.2.1**: Создать менеджер настроек
  - Сущность Setting с валидацией типов
  - Интерфейс SettingsServiceInterface
  - Use Case SettingsUseCase для управления настройками
  - Файловый сервис FileSettingsService с сохранением в JSON
  - Контроллер SettingsController
  - DTO для передачи данных

- [x] **TASK-2.2.2**: Адаптер настроек
  - Консольный адаптер для отображения настроек
  - Интерактивное меню изменения настроек
  - Валидация вводимых значений
  - Интеграция с главным меню

### 2.3 Система сохранений
- [x] **TASK-2.3.1**: Создать репозиторий сохранений
  - Валидация формата сохранений
  - Методы для CRUD операций
  - Файловая реализация с JSON

- [x] **TASK-2.3.2**: Адаптер загрузки игры
  - LoadGameMenuAdapter
  - Список доступных сохранений
  - Предпросмотр персонажа
  - Интеграция с главным меню

## Результат этапа
Полноценное главное меню с настройками, управлением сохранениями и навигацией ко всем игровым модулям.
