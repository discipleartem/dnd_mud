# Этап 2: Главное меню

## Обзор этапа
Разработка полнофункционального главного меню с навигацией по всем основным модулям игры.

## Цели этапа
- Создать гибкую систему меню с цветовой схемой
- Реализовать управление настройками игры
- Настроить систему сохранений и загрузки игр

## Задачи этапа

### 2.1 Базовая система меню
- [ ] **TASK-2.1.1**: Создать базовый класс меню
  - AbstractMenu с навигацией
  - MenuRenderer для отрисовки
  - Цветовая схема элементов меню

- [ ] **TASK-2.1.2**: Реализовать главное меню
  - MainMenuAdapter с пунктами меню
  - Обработка выбора пользователя
  - Переходы к соответствующим модулям

### 2.2 Система настроек
- [ ] **TASK-2.2.1**: Создать менеджер настроек
  - SettingsManager для загрузки/сохранения
  - Поддержка языка и hardcore режима
  - Валидация настроек

- [ ] **TASK-2.2.2**: Адаптер настроек
  - SettingsMenuAdapter
  - Интерфейс изменения настроек
  - Применение настроек в реальном времени

### 2.3 Система сохранений
- [ ] **TASK-2.3.1**: Создать репозиторий сохранений
  - SaveGameRepository (JSON файлы)
  - Валидация формата сохранений
  - Методы для CRUD операций

- [ ] **TASK-2.3.2**: Адаптер загрузки игры
  - LoadGameMenuAdapter
  - Список доступных сохранений
  - Предпросмотр персонажа

## Результат этапа
Полноценное главное меню с настройками, управлением сохранениями и навигацией ко всем игровым модулям.
