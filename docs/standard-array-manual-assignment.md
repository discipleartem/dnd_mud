# Ручное распределение стандартного набора характеристик

## Обзор

Теперь при выборе метода генерации "Стандартный набор" пользователь может самостоятельно распределять значения (15, 14, 13, 12, 10, 8) по характеристикам персонажа.

## Процесс распределения

1. **Выбор метода** - Пользователь выбирает "Стандартный набор"
2. **Пошаговое распределение** - Для каждой характеристики пользователь выбирает значение из доступных
3. **Валидация** - Система проверяет, что все значения использованы корректно
4. **Подтверждение** - Показывается финальный результат распределения

## Новые методы

### AttributeGenerator

- `get_standard_array_values()` - Возвращает доступные значения для распределения
- `validate_standard_array_assignment(attributes)` - Проверяет валидность распределения

### CharacterBuilder

- `set_attributes_standard_array_manual(attributes)` - Устанавливает характеристики с валидацией

### CharacterCreationMenu

- Улучшен `_handle_standard_array()` для интерактивного распределения
- Добавлены вспомогательные методы отображения

## Преимущества

- **Больше контроля** - Игрок решает, какие характеристики важнее
- **Стратегический выбор** - Можно оптимизировать под класс персонажа
- **Интуитивный интерфейс** - Пошаговый процесс с подсказками
- **Валидация** - Защита от ошибок при распределении

## Пример использования

```
Доступные значения: 15, 14, 13, 12, 10, 8

Нераспределенные характеристики:
1. STR (strength)
2. DEX (dexterity)  
3. CON (constitution)
4. INT (intelligence)
5. WIS (wisdom)
6. CHA (charisma)

Выберите характеристику: 1
Доступные значения для STR:
  15 (+2)
  14 (+2)
  13 (+1)
  12 (+1)
  10 (+0)
  8 (-1)

Выберите значение для STR: 1
✓ STR установлен на 15
```

## Валидация

Система проверяет:
- Все 6 характеристик распределены
- Использованы ровно значения из стандартного набора
- Нет дублирования значений
