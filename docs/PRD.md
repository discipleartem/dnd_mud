## Product Requirements Document

**Версия:** 2.0

## 1. Обзор проекта

### 1.1 Концепция
Однопользовательская консольная (терминальная) текстовая игра по мотивам механики настольной игры Dungeons & Dragons 5-й редакции.

### 1.2 Целевая аудитория
- Игроки настольных RPG игр
- Любители текстовых приключений
- Поклонники D&D 5e механики

### 1.3 Ключевые особенности
- Полностью консольный интерфейс
- Аутентичная механика D&D 5e (в будущем)
- Простая и понятная навигация
- Модульность системы с поддержкой модов
- Чистая архитектура
- Полная локализация (i18n) через YAML
- Модульная система приключений

## 3. Архитектурные принципы

### 3.1 Технологический стек
- **Язык программирования:** Python 3.12+
- **Формат данных:** YAML (игровые данные, локализация), JSON (сохранения)
- **Конфигурации:** YAML
- **Локализация:** YAML с полной поддержкой i18n

### 3.2 Архитектурные принципы
- **Zen Python:** Простое лучше сложного, читаемость важна
- **KISS:** Максимально простые решения
- **YAGNI:** Только необходимая функциональность
- **DRY:** Никакого дублирования кода
- **Чистая архитектура:** Чёткое разделение слоёв
- **Модульность:** Все системы работают через API и YAML
- **Локализация:** Все текстовые строки вынесены в YAML

### 3.3 Структура слоёв
```
src/
├── entities/      # Бизнес-сущности (Character, Adventure, I18n)
├── use_cases/     # Сценарии использования (GameUseCase, I18nManager)
├── ui/           # Интерфейс пользователя (Console)
├── core/         # Координация (Game)
├── interfaces/   # Абстракции (UserInterface, I18nAPI)
└── tools/        # Инструменты (AdventureEditor)
```

### 3.4 Структура данных
```
data/
├── i18n/         # Языковые файлы локализации
├── adventures/  # Модули приключений
├── characters/   # Данные персонажей
├── mechanics/    # Игровые механики
└── saves/        # Сохранения игры
```
