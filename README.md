# Текстовая MUD-игра

Текстовая однопользовательская MUD-подобная игра на чистом Python 3.13.

## Особенности
- Чистый Python 3.13+ с минимальными зависимостями
- Текстовый интерфейс с использованием Rich для оформления
- Двойная система ввода: меню и текстовые команды
- Сохранение игры в JSON
- Конфигурация мира через YAML
- Кроссплатформенность (Windows/Linux)
- Сборка в исполняемые файлы через PyInstaller
