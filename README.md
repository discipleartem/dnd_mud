# D&D Text MUD 5e

Однопользовательская консольная текстовая игра по мотивам Dungeons & Dragons 5-й редакции.

## Описание

D&D Text MUD 5e - это простая консольная ролевая игра, реализующая базовые механики Dungeons & Dragons 5-й редакции. Проект следует принципам KISS, YAGNI и Zen of Python для создания понятного и поддерживаемого кода.

## Возможности

- ✅ Простое главное меню
- ✅ Создание персонажа (имя, раса, класс)
- 🚧 В разработке: Расширенные характеристики
- 🚧 В разработке: Приключения
- 🚧 В разработке: Сохранение/загрузка

## Структура проекта

```
dnd_mud/
├── main.py              # Точка входа
├── game.py              # Основной игровой цикл
├── character.py          # Модель персонажа
├── data/               # YAML данные
│   ├── races.yaml
│   ├── classes.yaml
│   └── backgrounds.yaml
└── docs/               # Документация
```

## Используемые технологии

- **Python 3.12+** - основной язык разработки
- **PyYAML** - работа с данными в YAML формате
- **Стандартная библиотека** - логирование, работа с файлами

## Применяемые принципы

- **KISS (Keep It Simple, Stupid)** - простота решений
- **YAGNI (You Ain't Gonna Need It)** - только нужная функциональность
- **Zen of Python** - читаемость и явность
- **DRY (Don't Repeat Yourself)** - избегание дублирования

## Установка и запуск

### Требования

- Python 3.12 или выше
- pip (для установки зависимостей)

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/discipleartem/dnd_mud.git
cd dnd_mud

# Создание виртуального окружения
python3 -m venv .venv
source .venv/bin/activate

# Установка зависимостей
pip install -e .
```

### Запуск

```bash
# Активация виртуального окружения
source .venv/bin/activate

# Запуск игры
python main.py
```

## Тестирование

```bash
# Запуск всех тестов
pytest tests/ -v

# Запуск конкретного теста
pytest tests/test_welcome.py -v

# Запуск с покрытием
pytest --cov=src tests/
```

## Разработка

### Качество кода

Проект следует строгим стандартам качества:

- **Type hints** во всех функциях и методах
- **Docstrings** в Google стиле на русском языке
- **PEP8** соблюдение стиля кода
- **Unit тесты** для всех компонентов

### Линтинг и форматирование

```bash
# Проверка кода
ruff check src/

# Автоматическое исправление
ruff check --fix src/

# Проверка типов
mypy src/
```
