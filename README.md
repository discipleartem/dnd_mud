# D&D Text MUD

Однопользовательская консольная текстовая игра по мотивам механики настольной игры Dungeons & Dragons 5-й редакции.

## 🎯 Обзор

D&D Text MUD — это консольная RPG игра, которая полностью реализует механику Dungeons & Dragons 5e в текстовом формате. Игра включает систему модификаций, локализацию и полноценный процесс создания персонажа.

## ✨ Ключевые особенности

- **🎮 Полностью консольный интерфейс** — Beautiful terminal UI с Rich
- **⚔️ Аутентичная механика D&D 5e** — Полная реализация правил
- **🔧 Система модификаций** — Расширение контента через YAML файлы
- **🌍 Локализация (i18n)** — Поддержка русского и английского языков
- **💾 Сохранение персонажей** — Экспорт/импорт в JSON формате
- **📖 Обучающие приключения** — Tutorial для новичков

## 🚀 Быстрый старт

### Требования

- Python 3.12+
- Виртуальное окружение (рекомендуется)

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/discipleartem/dnd_mud.git
cd dnd_mud

# Создание виртуального окружения
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или
.venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -e .
```

### Запуск игры

```bash
# Запуск через CLI
dnd-mud

# Или напрямую
python -m dnd_mud.main
```

## 🎮 Игровой процесс

### Создание персонажа

1. **Выбор расы** — Человек, Эльф, Полуорк с подрасами
2. **Генерация характеристик** — Стандартный массив, Point Buy, или случайная генерация
3. **Выбор класса** — Воин, Паладин, Плут, Бард, Жрец
4. **Предыстория** — Определяет начальные навыки и снаряжение
5. **Навыки и языки** — На основе расы, класса и предыстории
6. **Заклинания и снаряжение** — Начальная экипировка персонажа

### Приключения

- **Начало Пути** — Обучающее приключение
- **Основной сюжет** — Эпическая история с побочными квестами
- **Диалоги и проверки** — Интерактивное взаимодействие с миром
- **Репутация и фракции** — Система отношений с группами

### Боевая система

- Пошаговые бои с инициативой
- Атаки и повреждения по правилам D&D 5e
- Заклинания и классовые способности
- Тактический геймплей

## 🔧 Система модификаций

Создавайте собственный контент через YAML файлы в папке `/Mods/`:

### Структура мода

```
/Mods/my_mod/
├── mod_info.yaml      # Информация о моде
├── races.yaml         # Новые расы
├── classes.yaml       # Новые классы
├── backgrounds.yaml   # Новые предыстории
├── skills.yaml        # Новые навыки
├── languages.yaml     # Новые языки
├── spells.yaml        # Новые заклинания
└── adventures.yaml    # Новые приключения
```

### Пример мода

```yaml
# mod_info.yaml
name: "My Custom Mod"
version: "1.0.0"
description: "Добавляет новые расы и классы"
author: "Modder"

# races.yaml
races:
  dragonborn:
    name: "Драконорожденный"
    bonuses:
      strength: 2
      charisma: 1
    special_traits:
      - "Драконья чешуя"
      - "Дыхательное оружие"
```

## 🧪 Разработка

### Настройка окружения

```bash
# Установка зависимостей для разработки
pip install -e ".[dev]"

# Настройка pre-commit hooks
pre-commit install
```

### Запуск тестов

```bash
# Все тесты
pytest

# С покрытием
pytest --cov=dnd_mud

# Быстрые тесты
pytest -m "not slow"
```

### Качество кода

```bash
# Форматирование
black src tests

# Проверка стиля
ruff check src tests

# Типизация
mypy src
```
